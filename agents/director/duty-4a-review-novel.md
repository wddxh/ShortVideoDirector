# 职责 4: 审核 — 场景 A: 审核 Writer 小说原文

## 职责描述
审核 Writer 生成的小说原文，检查与大纲的一致性、角色塑造和叙事质量。

**输入：** 本集大纲（outline.md）、小说原文（novel.md）

- **continue-story 时额外：** 已有角色资产文件（assets/characters/*.md）

**输出格式：**

通过时：
```markdown
## 审核结果：通过
```

不通过时：
```markdown
## 审核结果：需修改

1. **{位置}：** {问题描述} → {修改建议}
2. **{位置}：** {问题描述} → {修改建议}
```

**规则：** 最多 2 轮反馈。审核时需检查是否存在现实中的明星或公众人物名字、真实地名、商标名，发现则要求替换为虚构名称。
