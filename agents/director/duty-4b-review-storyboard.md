# 职责 4: 审核 — 场景 B: 审核 Storyboarder 分镜

## 职责描述
审核 Storyboarder 生成的分镜，检查叙事完整性、节奏、台词密度和技术合规性。

### 输入

小说原文（novel.md）、分镜（storyboard.md）、本集大纲（outline.md）

- **continue-story 时额外：** 已有角色资产文件（assets/characters/*.md）

### 输出格式

通过时：
```markdown
## 审核结果：通过
```

不通过时：
```markdown
## 审核结果：需修改

1. **镜头 {N}：** {问题描述} → {修改建议}
2. **镜头 {N}：** {问题描述} → {修改建议}
```

### 规则

重点审核叙事完整性、集尾收束、剧情节奏、人物言行与性格一致性、台词频率和声音连续性（避免长时间无台词或无声音）、信息传达是否按大纲标注自然执行（禁止生硬旁白灌输）、剧情铺垫是否充分、主角内心独白是否足够、单镜头内容密度（检查每个镜头的剧情事件、台词数量、信息传达量是否与该镜头时长相匹配，避免短时长镜头内塞入过多内容导致节奏过于紧凑，也避免长时长镜头内容空洞）、台词时长匹配（台词量是否与时间段时长自然匹配，避免语速过快）、画面文字（是否存在需要视频 AI 渲染的具体文字或数字，应改为角色台词传达）、版权规避（是否存在现实中的明星或公众人物名字、真实地名、商标名，发现则要求替换为虚构名称）、镜头间连贯性（相邻镜头之间角色动作/姿势/表情/场景是否有不合理的突变）
